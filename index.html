Vírus

programa malicioso projetado para se espalhar de um computador para outro, causando danos, roubo de dados ou interrompendo o funcionamento normal do sistema.

Bugs

Um bug pode permitir que um invasor contorne as medidas de autenticação e ganhe acesso não autorizado a informações confidenciais. Outros bugs podem ser explorados para executar códigos maliciosos, causar falhas no sistema, ou permitir ações que deveriam ser restritas.

Ramsoware

O "ransomware" é um tipo de software malicioso (malware) que tem como principal objetivo criptografar arquivos ou bloquear o acesso a sistemas e dispositivos, exigindo um pagamento (geralmente em criptomoedas) para restaurar o acesso ou descriptografar os arquivos afetados.
